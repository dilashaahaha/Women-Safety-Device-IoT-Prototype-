# Safety Bracelet IoT Emergency Alert System

An IoT-based wearable safety device that sends emergency SMS alerts with live GPS location when activated.

## Features

* SOS Emergency Button
* GPS Location Tracking
* GSM SMS Notifications
* Google Maps Location Link
* LED & Buzzer Alerts

## Hardware Used

* ESP32
* SIM800L GSM Module
* GPS Module
* Push Button
* Piezo Buzzer
* LED

## Software Used

* Arduino IDE
* Embedded C++
* TinyGPS++ Library

## Project Objective

To provide a simple, portable, and reliable emergency alert system for personal safety.
